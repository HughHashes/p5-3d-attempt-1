Welcome, Dimensional Traveller

This addon to p5.js is intended to extend the vector functions to any number of dimensions. The example below shows the addition of two points in 4 dimensions, but you're not limited to 4, you can go as many as you like! As long as it is less than 52.Try changing the numbers and seeing the output.







0.00 0.00 0.00 0.00

The function used to do the calculations above looks like this: 

result = nAdd(pos1, pos2);
And the vectors can be created like this: 
var pos1 = nVector(3, 1, 4, 1);
Currently I believe all the standard vector functions in p5 can be used in the same way using this addon, but if you find a bug please let us know at our GitHub repo.
